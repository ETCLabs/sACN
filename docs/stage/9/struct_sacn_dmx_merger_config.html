Detailed Description

A set of configuration information for a merger instance.

Field Documentation

◆ slot_owners

sacn_source_id_t* slot_owners

Buffer of DMX_ADDRESS_COUNT source IDs that indicate the current winner of the merge for that slot, or #DMX_MERGER_SOURCE_INVALID to indicate that no source is providing values for that slot.

You can use SACN_DMX_MERGER_SOURCE_IS_VALID() if you don't want to look at the slot_owners directly. Memory is owned by the application.

◆ slots

uint8_t* slots

Buffer of DMX_ADDRESS_COUNT levels that this library keeps up to date as it merges.

Memory is owned by the application.

◆ source_count_max

size_t source_count_max

The maximum number of sources this merger will listen to.

May be SACN_RECEIVER_INFINITE_SOURCES. This parameter is ignored when configured to use static memory – SACN_DMX_MERGER_MAX_SOURCES_PER_MERGER is used instead.
